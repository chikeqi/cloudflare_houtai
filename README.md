
## 🔧 配置说明

### 环境变量/KV 绑定

| 绑定名 | 类型 | 说明 |
|--------|------|------|
| `NAV_KV` | KV 命名空间 | 存储所有数据（必需） |

### 默认管理员密码

- 用户名：`admin`（固定）
- 密码：`admin123`（首次登录后请修改）

### KV 数据结构（自动创建，无需手动操作）

| Key | 说明 |
|-----|------|
| `sites` | 书签列表 |
| `blog_posts` | 文章列表 |
| `admin_password` | 管理员密码 |
| `site_title` | 站点标题 |
| `site_subtitle` | 站点副标题 |
| `site_logo` | Logo URL |
| `site_logo_link` | Logo 跳转链接 |
| `header_bg` | 页眉背景图 |
| `cn_link` | 国内线路链接 |
| `views:文章ID` | 文章阅读量 |
| `img:文件名` | 上传的图片 |
| `session:token` | 登录会话 |

## 🛠️ 日常使用

### 发布文章

1. 登录后台 `/admin`
2. 填写标题、分类、内容
3. 可选：封面图、标签、摘要
4. 选择状态（发布/草稿）
5. 点击"发布文章"

### 管理书签

1. 登录后台 `/admin`
2. 填写名称、网址、分类
3. 可选：排序数字（越小越靠前）、Logo、描述
4. 点击"添加书签"

### 修改站点信息

1. 登录后台 `/admin`
2. 找到"站点设置"卡片
3. 修改标题、副标题、Logo 等
4. 点击"保存设置"

### 修改密码

1. 登录后台 `/admin`
2. 点击右上角"🔑 修改密码"
3. 输入原密码和新密码
4. 确认修改

## 📝 注意事项

1. **KV 绑定名必须是 `NAV_KV`**，区分大小写
2. **不要删除 `functions` 目录**（如果有的话），或者确认只有 `_worker.js`
3. 图片上传限制 5MB，会自动存入 KV
4. 书签排序：数字越小越靠前
5. 置顶文章会显示在列表最前面
6. 国内线路按钮：只有在后台填写了链接才会显示

## ❓ 常见问题

### Q: 首页显示"暂无文章"？
A: 需要先在后台发布文章，并确保状态为"发布"。

### Q: 图片上传失败？
A: 检查图片大小是否超过 5MB，或格式是否为图片格式。

### Q: 富文本编辑器不显示？
A: 检查网络是否能访问 unpkg.com，或更换 CDN 源。

### Q: 忘记管理员密码？
A: 在 Cloudflare KV 中直接修改 `admin_password` 的值。

### Q: 如何绑定自定义域名？
A: Pages 项目 → **"自定义域"** → **"设置自定义域"**，按提示添加 DNS 记录。
